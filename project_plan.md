# Highbarcode B2B Website

## 1. Project Description
Highbarcode is a professional B2B company specializing in industrial barcode printing systems, technical sales, and hardware maintenance. The website serves as a digital presence to showcase expertise in Zebra, Honeywell, Bixolon, TSC printer maintenance, sensor calibration, and custom label software development. Target audience includes enterprise procurement teams, IT managers, and technical directors at major corporations, university hospitals, and international medical institutions.

## 2. Page Structure
- `/` - Home (Hero, Services, Portfolio, Contact)

## 3. Core Features
- [ ] Hero section with company introduction and CTA
- [ ] Services overview with 3 detailed service cards
- [ ] Portfolio section with NDA-protected project descriptions
- [ ] Contact form for business inquiries
- [ ] Navigation with external shop/store link
- [ ] Korean language localization throughout

## 4. Data Model Design
No database required for this phase. Contact form will use Readdy form submission.

## 5. Backend / Third-party Integration Plan
- Supabase: Not required for initial phase
- Shopify: Not connected (external shop link only)
- Stripe: Not required
- Forms: Readdy form submission for contact inquiries

## 6. Development Phase Plan

### Phase 1: Homepage Development
- Goal: Build complete homepage with all 4 sections (Hero, Services, Portfolio, Contact)
- Deliverable: Fully functional, animated, Korean-language homepage with navigation and footer

### Phase 2: Enhancement (Future)
- Goal: Add additional pages (About, detailed service pages, blog)
- Deliverable: Expanded site with deeper content
